Algorithm Visualization

There are currently three types of algorithm visualization you can make.
1. Finding Closures
2. Cross Joining Tables
3. Natural Joining Tables

Finding Closures

                
    const attributeSet = "C"
    const FDSet = ["AC->F", "CEF->B", "C->D", "DC->A"]

    sql.animationClosure(attributeSet, FDSet, 500, "Closure")

Cross Joining Tables

                
    const tableCJ1 = {
        "attributes": ["A", "B", "C"],
        "data": [
            { "A": 1, "B": 2, "C": 3 },
            { "A": 4, "B": 5, "C": 6 },
            { "A": 7, "B": 8, "C": 9 },
        ]
    }

    const tableCJ2 = {
        "attributes": ["C", "D", "E"],
        "data": [
            { "C": 1, "D": 2, "E": 3 },
            { "C": 3, "D": 4, "E": 5 },
            { "C": 6, "D": 7, "E": 8 },
        ]
    }

    sql.animationCrossJoin(tableCJ1, tableCJ2, 500, "CrossJoin")
