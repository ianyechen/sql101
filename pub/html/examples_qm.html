Quiz Types

There are currently two types of quizzes you can make.
1. Multiple Choice
2. Make Instance

Multiple Choice

A multiple choice styled quiz where the user can navigate freely between questions. There will be feedback upon the user selection an answer and submitting it.
                
    const tableMC = {
        "attributes": ["sID", "oID", "grade"],
        "data": [
            { "sID": 1, "oID": 2, "grade": 85 },
            { "sID": 1, "oID": 3, "grade": 30 },
            { "sID": 2, "oID": 3, "grade": 47 },
            { "sID": 2, "oID": 4, "grade": 68 },
            { "sID": 3, "oID": 4, "grade": 95 },
            { "sID": 4, "oID": 4, "grade": 34 },
            { "sID": 4, "oID": 5, "grade": 68 },
            { "sID": 5, "oID": 6, "grade": 64 },
            { "sID": 6, "oID": 6, "grade": 86 },
            { "sID": 7, "oID": 7, "grade": 94 },
            { "sID": 8, "oID": 9, "grade": 36 },
            { "sID": 9, "oID": 10, "grade": 44 },
        ]
    }

    const problemsMC = [
        {
            "question": "If we do SELECT avg(grade) FROM RELATION GROUP BY sID, how many rows will be in the resulting relation?",
            "choices": ["0 rows", "5 rows", "-2 rows", "9 rows"],
            "answer": "9 rows",
        },
        {
            "question": "If we do SELECT max(grade) FROM RELATION, what would be in the resulting relation?",
            "choices": ["e^2", "3.14", "95", "0"],
            "answer": "95",
        },
        {
            "question": "If we do SELECT avg(grade) FROM RELATION WHERE sID = 1, what would be in the resulting relation?",
            "choices": ["57.5", "ln(5)", "-100", "0"],
            "answer": "57.5",
        },
    ]

    sql.createMCQuiz(problemsMC, tableMC, "MC")

Make Instance

A quiz where the user needs to make an instance of the table based on the question. The user can navigate freely between questions. There will be feedback upon the user selection an answer and submitting it.
                
    const problems = [
        {
            "question": "Given the following FDs, make an instance of the relation that violates the set of FDs. S = { A -> BC, B -> D }",
            "FDSet": ["A->BC", "B->D"]
        },
        {
            "question": "Given the following FDs, make an instance of the relation that violates the set of FDs. S = { BC -> D, AE -> B }",
            "FDSet": ["BC->D", "AE->B"]
        },
        {
            "question": "Given the following FDs, make an instance of the relation that violates the set of FDs. S = { A -> B, C -> D, B -> E }",
            "FDSet": ["A->B", "C->D", "B->E"]
        },
    ]

    const table = {
        "attributes": ["A", "B", "C", "D", "E"],
    }

    sql.createMakeInstanceQuiz(problems, table, "MI")

Quiz Options

There are currently two options that you can add to the quizzes.
1. With Performance
2. With Time

With Performance

The performance option allows for the score for the quiz to be tracked. The user is no longer able to navigate freely between questions. The user moves on to the next question only if the user has submitted their answer.
